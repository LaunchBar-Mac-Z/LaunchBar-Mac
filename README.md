## Overview

LaunchBar Professional Suite is a complete keyboard productivity platform with indexing depth, scripting capability, and customization for power-user workflows.

Custom index rules control which files, folders, and data sources LaunchBar indexes, enabling focused results that exclude irrelevant content. Index templates handle specialized data sources including application-specific content and custom databases.

Script actions in AppleScript, JavaScript, Swift, Python, and shell execute custom operations triggered through the LaunchBar interface. Script actions accept input from the search field and produce output that feeds into subsequent actions, enabling sophisticated multi-step workflows.

Instant Send passes the currently selected content in any application to LaunchBar for processing through any action, enabling context-sensitive operations on content without copying it to the clipboard first.

LaunchBar Professional Suite is the keyboard productivity platform for power users who want the fastest possible interface between intent and execution across every aspect of their computing workflow.

## Performance & Integration

LaunchBar is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while LaunchBar processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
